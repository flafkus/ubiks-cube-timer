# Ubik's Cube Timer
A simple timer for the Ubik's Cube in Hypixel Skyblock, great for grinding Motes in the Rift.

Made this for 1.21.5, not sure if it works in other versions, can't be bothered to check.

## Features

- Tracks whenever you complete a Split or Steal game in the Rift & starts the 2 hour timer
- Saves the timer during game restarts
